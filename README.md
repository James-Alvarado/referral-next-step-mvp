# referral Next Step MVP

##Project Overview

Referral Next Step is a simple MVP web app that helps patiens understand what to do after a referral is approved. the Goal is to reduce confusion, improve follow-up, and guide patients toward the next step in their care. 

##Problem

In the U.S., 20% to 50% of medical referrals are never completed — meaning up to 1 in 2 patients falls through the cracks. Even in strong systems like NYC, about 1 in 4 patients does not follow through.

##Solution 

This MVP provides a simple and clear referral follow-up experience. The user selects a specialist type, insurance plan, and checks the referral status. After that, the app takes the user to a new page where they can immediately see: 
- available in-network providers
- available dates to schedule apointment directly from the app 
- a reminder of what to bring to the appointment

The goal is to reduce confussion, improve follow-up, and help patients move foward in their care with fewer manual steps.  

## MVP Scope

This project is intentionally scoped as a interactive MVP. The current version includes:

- a referral check page
- specialist type selection
- insurance plan selection 
- approved referral status flow
- navigation to a second page
- avalilable in-network provider options
- available appointment date options 
- appointment reminder

Out of scope for this version:
- real insurance verification
- live provider network data
- real appointment shceduling backend
- patient login
- live hospital system integration
